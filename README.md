# (C# 코딩) 간단 계산기

## 개요
- C# 프로그래밍 학습
- 1줄 소개: 간단한 사직연산을 할 수 있는 계산기
- 사용한 플랫폼:
  -C#, .NET Windows Forms, Visual Studio, GitHub
- 사용한 컨트롤:
  - Label, TextBox, Button
- 사용한 기술과 구현한 기능:
  - Visual Studio를 이용하여 UI 디자인
  - 숫자 버튼 입력 및 연산자 버튼 입력
  - `DataTable.Compute`를 활용한 여러 연산 처리
  - `CE`, `C`, `DEL` 버튼으로 입력 초기화 및 삭제 기능
